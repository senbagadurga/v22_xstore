Zebra now owns several of the scanner brands such as Symbol.
The core scanner driver must be installed for the Zebra scanner driver.

For 64 bit Windows unzip Zebra_CoreScanner_Driver_(64bit).zip in the windows_64 directory to a directory of 
your choosing and execute Zebra CoreScanner Driver executable within the directory. 

For 64-bit Linux install the zebra-scanner-corescanner rpm in the linux_64 directory.
